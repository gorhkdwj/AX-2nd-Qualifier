### 2-3. 무신사

**Codex 보고서에서 검증된 내용**
- 2025년 사상 최대 실적·수출 급증: 공식 뉴스룸 원문과 일치 `[검증됨]`.
- **물류 자회사 적자**: 무신사로지스틱스 2021년 이후 4년 연속 적자, **2025년 순손실 98억**, 풀필먼트 내재화+배송 외주(CJ대한통운 등)로 고정비·외주비 이중부담(외주용역비 2024년 238억→2025년 305억). 비즈니스포스트 원문과 정확히 일치 `[검증됨]`.

**독립 조사로 추가 확인된 신규 사실**
- **상품 혼용률/표기 오류의 반복** `[검증됨/교차확인]`: 2024년 12월~2025년 1월 다운·캐시미어 **7,968개 상품 전수조사에서 42개 브랜드·165개 상품 오기재 적발**(무신사 뉴스룸 2025-0123-2, 이투데이) → 이후 **2025년 11~12월 노스페이스 패딩 13개 스타일(28 SKU)에서 혼용률 오기 재발**, 대행사 벌점 50점·선제 전액 환불(연합뉴스TV 2025-11-28, 무신사 뉴스룸 2025-1202-2, 스카이데일리·산경투데이 등 다수 매체). → 재발 방지책에도 **1년 내 동일 유형 재발**이라는 구조적 리스크가 공개자료로 확인.
- 입점 셀러 수수료 갈등: 유통수수료가 20% 초반→30~35%로 상승해 백화점 수준 근접이라는 입점업체 불만 지속(한국섬유신문, 비즈한국) `[교차확인]`.
- 평판 이슈: 2026년 3월 오프라인 매장 외국인 응대 논란(경향신문), 2026년 5월 과거 광고 문구 재조명 논란·공식 사과(ZDNet, 무신사 뉴스룸 2026-0520-02) `[교차확인]`.

**주의할 점**
- Gemini가 서술한 "MD 육안 검수 전적 한정", "백오피스 대혼란", "비전 LLM 99% 인식률"은 **어떤 공개자료로도 뒷받침되지 않는 창작** → 사용 금지.

## 5. 사용자 최종 선정을 위한 참고 (중립 정리, 추천 아님)

각 기업을 선택할 경우 **공개자료로 검증된 문제**와 **플러그인 방향 후보**, **주의점**을 중립적으로 정리합니다. 어느 것이 최선인지는 판단하지 않습니다.

- **무신사**: (문제) 상품 혼용률·표기 오류 반복 / 물류 수익성. (플러그인 방향) 공개된 케어라벨 세분화 표기 기준에 맞춰 상품 상세정보 표기 정합성을 점검·경고하는 보조 도구. (주의) 실제 내부 검수·주문 데이터는 비공개 → 공개 기준·샘플 기반 MVP로 범위 한정 필요. 근거·구체성이 가장 두터움.
- **메디테라피**: (문제) ROAS·진정성 콘텐츠 부족. (플러그인 방향) 캠페인/랜딩 문구·제품 USP 입력 → 전환 병목·메시지·화장품 표시광고 리스크 점검. (주의) 실제 캠페인 데이터 비공개 → 샘플 기반 검증 위주. 부작용 이슈는 근거 부족하니 문제로 세우지 말 것.
- **마이리얼트립**: (문제) 예약/취소/환불 문의의 구조적 복잡성(+판결·환불 분쟁이라는 평판 리스크). (플러그인 방향) 공개 도움말 기반 문의 분류·근거 링크·답변 초안(RAG). (주의) 새로 확인된 판결·환불 분쟁은 플러그인으로 직접 풀 문제는 아님. CS 복잡성 자체는 codex 인용이 정확해 프로토타입화 용이.
- **채널톡**: (문제) 매출 고성장에도 완전자본잠식이라는 재무 리스크. (플러그인 방향) 공개 재무·SaaS 지표 기반 재무 건전성/런웨이 진단류 — 단 "고객이 겪는 문제" 프레이밍이 약함. (주의) 완전자본잠식 수치는 DART 원본 재확인 필수. 제품 자체(AI상담)는 채널톡의 기존 제품과 겹치지 않게 설계 필요.
- **삼일PwC**: (문제) AI 도입 초기 실무 부담(업계 공통). (플러그인 방향) 이기종 재무·조서 자료를 정규화하는 전처리 보조. (주의) 감사조서 기밀성으로 공개 데이터 실증이 가장 어렵고, 삼일 특정 근거가 얇음. 마감 내 완성 난이도 최상.

### 최종 선정 전 반드시 처리할 `[확인 필요]` 항목
1. 채널톡 완전자본잠식 → **DART에서 채널코퍼레이션 감사보고서 원본 확인**.
2. 메디테라피 최근 매출(성장 둔화) → 1차 자료(공시·IR·신뢰 매체) 확인.
3. 삼일PwC K-SOX 보고서 세부 문구·mk.co.kr 인터뷰 → 원문 확보 또는 근거에서 제외.
4. 마이리얼트립 대표 환불 고소 건 → 트래블아이 원문 접속해 사실관계 재확인.
5. 선택한 기업이 확정되면, 그 기업 문제의 모든 인용을 **원문 URL + 확인일**과 함께 `docs/company-selection.md`에 기록하고 `Decisionlog`에 선정 사유를 남길 것.
