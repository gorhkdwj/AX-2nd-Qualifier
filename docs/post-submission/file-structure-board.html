보드 사용법

노드를 선택하면 해당 파일군의 역할, 주요 경로, 의존 관계가 표시됩니다. 검색창과 필터로 특정 영역만 좁혀 볼 수 있습니다.

  • 파란 선: 기준 계약과 지침이 다른 파일을 규정합니다.
  • 초록 선: 스크립트나 입력이 결과물을 생성합니다.
  • 보라 선: 검증기가 fixture와 결과를 확인합니다.
  • 주황 선: 최종 제출 zip에 포함되는 흐름입니다.
  • 붉은 선: 로그와 작업 기록이 남는 흐름입니다.
