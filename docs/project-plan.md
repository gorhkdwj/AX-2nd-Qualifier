# 프로젝트 기획 · AX해커톤 예선 2차 제출(기업 미정)

## 문제 정의
- `docs/과제.md` 과제: 선택한 기업(또는 그 산업·고객)이 겪는, 공개 자료로 입증 가능한 실제 문제를 해결하는 Codex 플러그인을 만든다.
- 현재 단계: 어느 기업의 문제를 다룰지 아직 정해지지 않음 → 선정 절차는 `docs/company-selection.md` 참조.

## 목표와 성공 기준
- 후보 5곳(채널톡, 메디테라피, 무신사, 삼일PwC, 마이리얼트립) 중 근거·구현 가능성·차별성 기준으로 1곳 확정.
- 확정 기업의 문제를 공개 자료로 뒷받침하고, 이를 해결하는 Codex 플러그인(`src/.codex-plugin/plugin.json` + 스킬/코드)을 완성.
- `submission.zip` 구조(`src/`, `README.md`, `logs/`) 요건을 충족하고, 질문 5문항에 로그·플러그인과 정합성 있게 답변.
- 마감(2026-07-06 밤) 전 검증 완료.

## 범위
- 포함: 기업 선정 리서치, 문제 정의, Codex 플러그인 설계·구현, 로그 정리, 질문 5문항 작성, 제출 패키징.
- 제외: 카카오페이증권 관련 작업(별도 제출 완료), 여러 기업을 하나의 제출물로 묶는 것(규정상 불가).

## 주요 사용자와 사용 시나리오
- 해커톤 심사위원/주최측이 `submission.zip`을 열어 `plugin.json`·스킬·로그·README·질문 답변의 정합성을 확인하며 평가.
