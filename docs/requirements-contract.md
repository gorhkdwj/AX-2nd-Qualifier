## 제출 구조 기준 (과제 규정 — 고정)
- `submission.zip` 루트: `src/`, `README.md`, `logs/`
- `src/.codex-plugin/plugin.json` 필수
- `plugin.json` 외에 실제 동작 구성요소 1개 이상 필요: `src/skills/<이름>/SKILL.md`(가장 일반적) 및/또는 `.mcp.json` 및/또는 직접 작성한 실행 코드
- `logs/` 는 AI와 주고받은 대화 로그 원본을 편집·발췌 없이 그대로 포함 (형식: md, txt, json, jsonl)
- 질문 5문항: 무엇을·누가·어떤 상황에서 쓰는가 / 왜 이 문제를 선택했는가 / 플러그인은 어떻게 작동하는가 / AI를 어떻게 활용했는가 / 어떻게 검증했는가

## 완료 기준
- 기업 1곳 확정 및 Decisionlog 기록
- 위 "제출 구조 기준"을 모두 충족하는 `src/`, `README.md`, `logs/` 존재
- 질문 5문항 답변이 로그·플러그인 내용과 모순 없음
- 근거로 사용한 모든 공개 자료가 출처(URL/제목/확인일)와 함께 기록됨
